Stick Wars Online

Welcome to Stick Wars, an engaging and strategic online game that puts you in command of stick figure armies battling for supremacy. 
Immerse yourself in the world of simple yet intense warfare, where resource management and tactical prowess are key to victory.

Gameplay:
1. Command Stick Armies: Take control of stick figure armies and lead them into epic battles.
2. Resource Management: Gather resources strategically to build and upgrade your forces.
3. Strategic Deployment: Plan your attacks and defenses wisely to outsmart your opponents.
4. Dynamic Battles: Experience fast-paced and dynamic battles in a stick figure world.

The game is written by using SFML library.

Getting Started:
1. Clone the repository: git clone https://github.com/sergeysolov/Stick-Wars-Online.git
2. Launch the game and start your stick figure conquest!

Multiplayer gaming:
Experience the thrill of Stick Wars with your friends through multiplayer gaming. 
To engage in battles together, establish a local network using one of the following methods:

1. Ethernet Connection: Connect your devices using Ethernet cables for a stable and direct connection.

2. Wi-Fi Connection: Utilize a local Wi-Fi network to seamlessly connect and enjoy multiplayer battles with friends.

3. Radmin WPN: Explore the option of using Radmin WPN, a program designed to establish a local network across the internet. This allows you to connect with friends remotely and engage in epic stick figure warfare.
